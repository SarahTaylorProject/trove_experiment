“A large number of the cases are from known outbreaks”.

This simply isn’t true. The vast majority of cases each day are from unknown sources, and there are currently more than 2000 cases that can’t be tracked or traced. This is double the number of known cases in Aged Care.

The number of daily cases has a strong correlation to the number of tests, with about a five day lag.

The average number of daily tests in July in Victoria was around 24,000. On 26 July, 43,000 tests were conducted which resulted in a new record of 723 cases 4 days later and 627 five days later. The following days had testing back around the 24,000, which resulted in lower new cases. Then there were 33,800 tests on 31 July, which resulted in yesterday’s new record of 725 (ie 5 days later).

The worry of this relationship is twofold.

The first is that more tests that are done, the higher the new cases. This means that Victoria doesn’t really know how bad things are. Ideally, the more tests done should not change the number of cases, as has been the case in NSW, which has been doing record testing, but with the essentially same number each day and nearly all from known sources. But it is really concerning when there is a relationship between increased testing and cases. It means that there is likely a lot more cases out there than we know about, which is revealed every time testing is increased. .

The second issue is this pattern of 4-5 day delay in seeing the impact of testing. If the impact of the stage 4 restrictions won’t kick in for 14 days, and add another 5 days to this to reflect the lag, then the number of new cases won’t start to reduce until around another 16 days time (unless there is reduced testing). This will be almost halfway into the 6 weeks of stage 4 and only gives a further 3 weeks to flatten the curve. Given there are so many cases, which will rise to probably over 1000 before it lowers, with a vast majority untracked, Victoria will be in lock down for a lot longer than 6 weeks

Stage 4 will work. But assuming new cases do peak at 1000 before they decline, it will be a long way back to single digits. Even if they have already peaked, which I suspect it hasn’t, it’s still a long way back from 725.

And even then, this assumes community compliance and enforcement of stage 4.

I disagree with the percentage of tests coming back positive being one of the most significant measures, which is why I don't typically mention it in my articles.

In those places the high testing percentage alludes to a higher incidence and a lot of cases not being picked up by routine testing, but in Victoria the daily test numbers have been reasonably consistent and the per capita rates are quite high.
